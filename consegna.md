# Consegna - L07 e L08 Ticketing PR

Questa repo serve per due momenti separati:

- L07: preparare il brief tecnico, senza modificare codice.
- L08: usare il brief per avviare una patch parziale e reviewabile.

Task comune:

```txt
Serve creare ticket dal supporto.
```

In questa repo `dal supporto` significa sorgente/canale della richiesta, non utente autenticato.

## Da Leggere

Segui i file in questo ordine:

1. `docs/istruzioni-l07.md`
2. `docs/istruzioni-l08.md`

Non saltare L07: L08 parte solo se hai gia' mappa dei punti di intervento, prompt patch limitato e reviewer leggero.

## Se Non Hai Completato L05-L06

Usa la cartella:

```txt
per-chi-non-ha-completato-l05-06/
```

Contiene artefatti minimi gia' prodotti per iniziare L07 senza bloccare il lavoro.

Se invece hai i tuoi output L05-L06, usa i tuoi: sono la fonte migliore.

## Regola Di Scope

Non copiare repo precedenti intere. Porta solo gli artefatti richiesti.

Non implementare feature complete: questa repo deve restare un esercizio su scope, gate, patch piccola e verifica manuale.
